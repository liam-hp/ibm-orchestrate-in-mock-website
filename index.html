üìã End Result

An aesthetically exact though functionally simple mocked-up version of your client's website with your WxO agent embedded to demonstrate value in a real-world context.

Option 1: Manual Upload

  1. Navigate to your client's website and take screenshots all the way down
  2. For best results, take screenshots between vertical spaces
  3. Click "Mock Website" and select "Manual Upload"
  4. Upload your screenshots (they will stack vertically)

  1. Create a free ScreenshotOne account (100 free screenshots/month)
  2. Copy your API key from the dashboard
  3. Click "Mock Website" and select "Auto Mockup"
  4. Enter the website URL and your API key
  5. Optionally check "Download screenshot" to save a copy

ü§ñ Embed Your Orchestrate Agent

Step 1: Get the Embed Script

  1. Deploy your agent in watsonx Orchestrate
  2. Option A (CLI): Run orchestrate channels webchat embed --agent-name=AGENT_NAME
  3. Option B (SaaS UI): Navigate to Agent Builder ‚Üí Channels ‚Üí Embedded Agent section
  4. Copy the HTML script tag

Step 2: Apply the Script

  1. Click "Embed Agent" button
  2. Paste the entire script tag into the textarea
  3. Click "Apply Script"

üí° Fullscreen Mode (Optional)

When embedding your agent, you can enable fullscreen mode by checking the "Enable Fullscreen Mode" checkbox. This mode:

  • Displays the chat interface as a fullscreen overlay
  • Hides the mock website background
  • Perfect for demos focused solely on the agent interaction
  • No need to use the Mock Website feature when fullscreen is enabled
